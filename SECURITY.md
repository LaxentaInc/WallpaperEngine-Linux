# Security Policy

## Supported Versions

The following versions of the Linux Wallpaper Engine are currently supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please report it via the GitHub Security Advisory feature.
We will review all reports within 48 hours and provide a patch within 7 days for critical issues.

*Note: Security patches are backported to 1.0.x.*
